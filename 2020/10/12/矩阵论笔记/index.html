3.1 线性映射与变换

3.1.1 线性映射概念与性质

定义1

$V_1, V_2$ 为数域$F$上两个线性空间，如果满足了可加性和齐次性，对于$T: V_1 \rightarrow V_2$既可以称作$T$是$V_1$到$V_2$的线性映射，某些书籍也记为$Hom(V_1, V_2)$。

实例
  • $\forall x\in V, f(x) = \eta_0, f(2x) \ne 2f(x)(\eta_0 \ne \theta)$ 非线性
  • $\forall x\in V, f(x) = x + \eta_0, f(2x) \ne 2f(x)(\eta_0 \ne \theta)$ 非线性
  • $A\in F^{n\times n}, f: F^{n\times n}\rightarrow F^{m\times m}, \forall x\in F^{n\times n} f(x) = xA$ 线性（常数矩阵也是一种线性加和）
  • $F_n[x]\rightarrow F_n[x], \forall p(x)\in F_n[x], f(p(x)) = p’(x)$ 线性
说明
  • $T(V_1)$ 显然是$T$的值域
  • $V_1 = V_2$ $T$ 收缩为变换
  • $T_\theta: V_1\rightarrow V_2, \forall \alpha\in V_1, \alpha\mapsto \theta_{V_2}$ ，$T_\theta$ 为零映射
  • $T_e: V\rightarrow V, \forall\alpha \in V, \alpha\mapsto\alpha $，$T_e$为恒等变换
  • $V_1 = V_2$时，映射收缩成变换​

定理1

$T:V_1\rightarrow V_2$，则：

  • $T(\theta_{V_1}) = \theta_{V_2}, T(-\alpha) = -T(\alpha)$
  • $\alpha_1,\alpha_2,…,\alpha_n$相关，那么$T\alpha_1,T\alpha_2,…,T\alpha_n$也相关（注意：$\alpha_i$组的无关推不出$T\alpha_i$的无关）
  • $dim R(T)\leq dim V_1$，满射时相等

定义2：映射的复合

有$T_1:V_1\rightarrow V_2$和$T_2:V_2\rightarrow V_3$，那么有$T_2T_1: T_2T_1(x) = T_2(T_1(x))$。显然$T_2T_1$是$T_1$和$T_2$的复合。

定理2

$W_1,W_2$是$V_1$的子空间，有：

  • $T(W_1+W_2)=T(W_1)+T(W_2)$
  • $T(W_1\cap W_2)\subset T(W_1)\cap T(W_2)$
实例：定理2.2

$A = \begin{pmatrix} 1&0\\0&0\end{pmatrix}\in R^{2\times2}, T:y=Ax,\forall x\in R^2,W_1=span\begin{pmatrix}1\\0\end{pmatrix},W_2=span\begin{pmatrix}1\\1\end{pmatrix}$，那么有：
$W_1\cap W_2 = \begin{pmatrix}0\\0\end{pmatrix},T(W_1\cap W_2)=\begin{pmatrix}0\\0\end{pmatrix},T(W_1)\cap T(W_2)=W_1$

利用几何语言说明：$A$是仿射变换，将$(x,y)$向$X$轴投影为$(x,0)$。$W_1$为$y=0$的任意向量，$W_2$为$y=x$上的任意向量，显然$W_1$和$W_2$的交点只有原点，做了仿射变换还是在零点；而$TW_1$还是维持原样，$TW_2$变成了$y=0$的任意向量，两者的空间重合了。

这个例子也间接说明了$\alpha_i$不相关，$T\alpha_i$也是可能相关的。
